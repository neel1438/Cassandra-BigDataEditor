Instructions to use our Tool


OVERVIEW

Our Web based Application allows the user to work with the Cassandra Database. The User can perform various operations such as Create, Drop, Retrieve Information about Keyspaces, Column Families and Clusters. Our tool provides the below mentioned Functionalities to the user to operate and perform various operations with the Cassandra Database.


  • KEYSPACES

    • Create - For Creating new Keyspace with the input specified replication factor.
    • Destroy - For Destroying an existing Keyspace.
    • List - For Listing the names of all existing keyspaces.


  • COLUMN FAMILIES

    • Create - For Creating new Column Family in the desired Keyspace.
    • Destroy - For Destroying an existing Column Family of the desired Keyspace.
    • List - For Listing the names of all the existing Column Families of the desired Keyspace.


  • DATA ENTRY

    • Add Entry - For Adding a new entry to the Column Family of desired Keyspace.
    • Remove Entry - For Removing an existing entry from the Column Family of desired Keyspace.


  • DATA SEARCH

    • Get Key - For Displaying the Key of the Column Family
    • Get Entries - For Getting the entries of the Column Family of desired Keyspace.


  • CLUSTER

    • Add Machine - For Adding a new machine to the Cluster.
    • Remove Machine - For Removing an existing machine from the Cluster.
    • List Machines - For Viewing the lists of all existing machines in a Cluster.
