Getting Started with our Tool

Steps to be Followed

  • First you need to install the required Libraries needed to use our tool ie, Cassandra, Pycassa (Python API) and Flask (Python Microframework based on WSGI).
  • Run the Cassandra Server using the command Cassandra -f on terminal (Make sure you have sudo access to the machine to run cassandra).
  • Download our Source Code from here .
  • From the Cassandra folder, run the python code.
  • Then point your browser to the localhost IP on Port 5000 (default port).
  • Now you can easily operate on the cassandra database using our web based application tool.

One can run this setup in just one step by running the script file named setup.sh from our source code.


For Adding a node to a Cluster you have to specify some inputs as mentioned in the input file ad Run the script addnode.sh from our source code on the machine to be added. (Make sure You have root access to the machine)



To know more about the functionalities being offered by our tool , one can refer to About the tool section
